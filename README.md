# YellowCatball.wxpr

A Preset pack for an old school program, Debugmode WAX.

For more infomation of the original Software, please read the YTPTennisWiki article because Wikipedia doesn't have one.
https://ytptennis.com/w/index.php?title=Debugmode_Wax

This preset contains at least 100 presets each, but there are 78 Plugin presets & 22 Transitions.

# How do i install?

We have an installer package now, please check the releases & download the installer, please remind that the installer is safe as usual.
